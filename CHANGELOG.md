# Changelog

This project uses semantic versioning once releases begin. Until the first tagged release, entries describe the current pre-release state on `main`.

## 0.1.0 - Unreleased

Initial pre-alpha CLI foundation.

Added:

- Python package scaffold with the `bible` console command.
- SQLite-backed Scripture storage model.
- Tiny ASV sample fixture for repeatable tests.
- Reference parsing for verses, chapters, and same-chapter verse ranges.
- Readable passage rendering with plain/no-color output.
- ASV bundle and local USFX import paths.
- Simple phrase search with optional book limiting.
- Book and chapter navigation helpers.
- Local JSON bookmarks and notes.
- Stacked comparison command.
- TUI planning placeholder and design notes.
- Release/install documentation and smoke-test checklist.
